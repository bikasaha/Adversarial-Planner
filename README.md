# POSTER: On the Adversarial Planning Capabilities of LLMs

## Overview

This repository contains artifacts used to evaluate the adversarial cyber-attack planning capabilities of Large Language Models (LLMs).

---

## Repository Structure

```yaml
├── data/
│   └── sample_organizational_scenario.txt   # Sample organizational context used for attack planning
└── prompts/
    ├── system_prompt.txt                   # Defines adversarial planner role and planning behavior
    └── T1_TEMPLATE.txt                     # User prompt template providing organizational context and planning instructions


```

## Notes

- Only one sample organizational scenario is included in this release.  
- The complete dataset will be made available upon acceptance.
